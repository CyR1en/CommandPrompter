# Planned Features

This document outlines the features and improvements planned for future releases of CommandPrompter v3.

## Internationalization (i18n)
The old internationalization system from v2 has been removed for now. We are currently working on building a much more robust and flexible localization system that will be introduced in an upcoming update.

## File-Based Prompt Builder
We are also planning to introduce a file-based prompt builder using JSON. This feature will allow you to design and configure complex prompts in dedicated JSON files, and then simply reference them within your commands. This approach will significantly clean up your command configurations and make sharing or reusing prompt designs much easier.
